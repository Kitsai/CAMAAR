# CAMAAR

Sistema para avaliação de atividades acadêmicas remotas do CIC

O sistema tem como objetivo melhorar a experiência de responder formularios
à respeito do semestre sendo possível avaliar as turmar.
Os formulários são criados por um usuário admin que,
geralmente, é o coordernador do curso e podem ser respondidos por Alunos e Professores.

O sistema garante anonimização das respostas
possuindo integração com os sistemas do SIGAA.
